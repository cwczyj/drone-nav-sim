# 农田边界坐标示例

## 示例1：矩形农田（成都地区）
```json
[[30.6742, 104.0667], [30.6742, 104.0687], [30.6762, 104.0687], [30.6762, 104.0667], [30.6742, 104.0667]]
```

## 示例2：不规则农田

```json
[[30.6720, 104.0650], [30.6730, 104.0660], [30.6740, 104.0655], [30.6735, 104.0640], [30.6725, 104.0645], [30.6720, 104.0650]]
```

## 示例3：大型农田
```json
[[30.6700, 104.0600], [30.6700, 104.0700], [30.6800, 104.0700], [30.6800, 104.0600], [30.6700, 104.0600]]
```

## 使用说明

1. **坐标格式**：每个坐标点表示为 `[纬度, 经度]`
2. **闭合多边形**：第一个点必须与最后一个点相同
3. **最少点数**：至少需要3个坐标点
4. **坐标顺序**：建议顺时针或逆时针方向排列点

## 在不同地图平台获取坐标

### Google Maps
- 右键点击地图上的位置
- 选择"这里有什么？"
- 坐标会显示在搜索框中
- 格式：纬度, 经度（例如：30.6742, 104.0667）

### 高德地图
- 点击地图上的位置
- 在右侧面板查看坐标
- 格式：经度, 纬度（注意：需要转换为 纬度, 经度）

### 百度地图
- 在地图上标记位置
- 点击"查看详情"
- 获取经纬度坐标

## 注意事项

- 不同地图平台的坐标系统可能有微小差异
- 建议使用相同的地图平台获取所有坐标点
- 对于大范围农田，建议使用更多坐标点以提高精度
