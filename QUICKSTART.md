# 快速入门指南

## 系统要求

- Python 3.8 或更高版本
- pip 包管理器

## 安装步骤

### 方法1: 使用安装脚本（推荐）

#### Linux/Mac:
```bash
cd drone-nav-sim
./install.sh
```

#### Windows:
```bash
cd drone-nav-sim
install.bat
```

### 方法2: 手动安装

```bash
# 安装后端依赖
cd backend
pip install -r requirements.txt

# 安装前端依赖
cd ../frontend
pip install -r requirements.txt
```

## 启动系统

### 一键启动

```bash
python run_all.py
```

首次运行时，系统会询问是否安装依赖，选择 `y` 即可。

### 手动启动

如果您想分别启动前后端服务：

#### 启动后端服务:
```bash
cd backend
uvicorn main:app --reload
```

后端服务将在 http://localhost:8000 启动

#### 启动前端服务（新终端）:
```bash
cd frontend
streamlit run app.py
```

前端服务将在 http://localhost:8501 启动

## 使用流程

### 1. 注册账号

1. 打开浏览器访问 http://localhost:8501
2. 点击"注册"标签
3. 填写用户名、邮箱和密码
4. 点击"注册"按钮

### 2. 登录系统

1. 切换到"登录"标签
2. 输入注册的用户名和密码
3. 点击"登录"按钮

### 3. 创建农田

1. 在侧边栏选择"农田管理"
2. 点击"创建新农田"
3. 填写农田信息：
   - 农田名称：例如"东北田块1"
   - 面积：例如 50.5 亩
   - 作物类型：例如"水稻"
   - 边界坐标：使用JSON格式，例如：
     ```
     [[30.6742, 104.0667], [30.6752, 104.0687], [30.6762, 104.0667], [30.6742, 104.0667]]
     ```
4. 点击"创建农田"

### 4. 查看农田可视化

1. 在侧边栏选择"农田可视化"
2. 所有农田将显示在地图上
3. 点击农田多边形可查看详细信息

## 边界坐标说明

边界坐标使用 JSON 数组格式，每个坐标点包含经纬度：

```
[[纬度1, 经度1], [纬度2, 经度2], [纬度3, 经度3], ...]
```

**注意事项：**
- 至少需要3个坐标点
- 建议使用4-6个点绘制较为准确的农田边界
- 第一个点应该与最后一个点相同，形成闭合多边形

**获取坐标的方法：**
1. 使用 Google Maps: 右键点击地图上的点，选择坐标
2. 使用 高德地图: 点击地图上的位置获取坐标
3. 使用 百度地图: 点击地图标记获取坐标

## 常见问题

### Q: 忘记密码怎么办？
A: 当前版本使用内存存储，可以重新注册账号。后续版本将支持密码重置功能。

### Q: 如何退出登录？
A: 在侧边栏点击"退出登录"按钮。

### Q: 数据会丢失吗？
A: 当前版本使用内存存储，重启服务后数据会丢失。后续版本将支持数据库持久化。

### Q: 可以同时创建多个农田吗？
A: 可以，登录后可以创建、编辑和删除多个农田。

### Q: 如何修改农田信息？
A: 在农田列表中，点击"编辑"按钮即可修改农田信息。
